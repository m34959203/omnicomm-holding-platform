# Контрибьютинг

## Конвенции

- **Коммиты:** Conventional Commits. Префикс на английском (`feat:`, `fix:`, `docs:`, `refactor:`, `test:`, `chore:`, `ci:`), описание — на русском, глагол в инфинитиве, без точки, ≤72 символа.
- **Ветки:** `feat/xxx`, `fix/xxx`.
- **Язык:** код и идентификаторы — английский; комментарии, докстринги, PR и issues — русский.

## Локальная проверка перед PR

```bash
pip install -r requirements.txt
python -m pytest -q          # тесты проходят
python -m compileall src     # синтаксис ок
ruff check src tests         # линт (если установлен ruff)
```

## Безопасность

- Никогда не коммитить `.env`, реальные логины/пароли Omnicomm, токены.
- В репозитории только `.env.example` с пустыми значениями.
- Токены и учётные данные не логировать (ТЗ §4.6).
