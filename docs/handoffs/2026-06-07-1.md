# Handoff 2026-06-07 №1 — боевой прогон по Горкомтрансу + анализ отчёта

Кому: Cowork. Задача: сделать **боевой прогон** отчёта по реальному парку
Горкомтранс (Караганда, вывоз ТКО, ~70 ТС) и дать **аналитику отчёта** уровня
CEO/консультанта по автопаркам.

## Как сделать боевой прогон (у тебя есть browser control)
Учётка Omnicomm в репо НЕ лежит (`.env`/`data/` gitignored), поэтому самый
простой путь — через **живую платформу** (клиент «Горкомтранс» уже настроен на
сервере, учётка хранится там):

1. Открой платформу: **https://sage-planets-footwear-align.trycloudflare.com**
   (если ссылка сменилась — спроси у Дмитрия актуальную trycloudflare).
2. Слева: **📊 Отчёты → Отчёт по клиенту → Горкомтранс**.
3. Период: **Прошлый месяц** (или Последние 30 дней). Цена топлива 320–341 ₸/л
   (можно «Подставить цену поставщика»). Тип ДТ: Лето.
4. Нажми **🚀 Сформировать отчёт** → скачай **.html** (и .pptx) → проанализируй.

Альтернатива (если есть доступ к серверу `/home/ubuntu/omnicomm-fleet-report` с `.env`):
```bash
python -m omnicomm_report --source api --preset last-month \
    --fuel-price 320 --html --xlsx --with-track
# результат в output/
```

## Что проанализировать (как CEO)
- KPI парка: пробег, расход, стоимость ₸, доля холостого хода.
- Перерасход/экономия по нормам (₸), топ проблемных ТС (скоринг), сигналы.
- Работа на погрузке: л/моточас, точки погрузки (GPS) — оцени реалистичность.
- Классификация техники — корректны ли типы (мусоровозы/спецтехника).
- Слабые места данных (тёмные ТС, нет датчиков надстройки, нет геозон).
- Дай вывод: где деньги (десятки млн ₸/год?), что приоритизировать.
