## 1. Механика раздела

`/acreports` — конструктор отчётов: слева общая панель параметров, справа вкладки отчётов.

**Общие параметры (на все отчёты):**
- **Период:** пресеты `сегодня`/`вчера`/`7 дней`/`месяц` + произвольный диапазон с календарём.
- **Выбор ТС:** дерево по группам/компаниям (~2000 ТС), поиск, фильтр по статусу
  (зелёный/оранжевый/красный/серый «нет данных»/все), «Выделить все», счётчик выбранных.
- **Геозоны:** вторая вкладка панели (для отчёта «Посещение геозон»).

**Выход/экспорт (табличные):** правый клик → Обновить / Печать (PDF) / **Экспорт в xls** / Помощь.
Пагинация (50/100/300/500), кнопка **«Зафиксировать»** (снапшот), несколько отчётов вкладками,
у части — переключаемые группы колонок и выбор конкретного ТС. Карта-отчёты (Трек,
Местоположение) — карта Omnicomm, без xls.

## 3. Описание форм

**1. Выдача топлива** — суммарный объём выдачи за период. Кол.: №, ТС, время начала/окончания, объём л.

**2. Выдачи/заливы/сливы (только АТЗ)** — №, ТС, время, событие, объём л. На обычных ТС — «доступен только для АТЗ».

**3. Движение** (по 1 ТС) — сводка: длина пути, время перемещения %, ср.скорость, время стоянок %,
отсутствие данных %. Кол.: №, Тип (Движение/Стоянка/Смена места/Нет данных), Длительность, время начала/окончания, длина пути, ср.скорость.

**4. Журнал** (по 1 ТС) — сырой лог пакетов (~30с интервал), ~70 колонок: дата/время, ключ, GPRS/GPS,
питание В, спутники, координаты, направление, высота, пробег, скорость GPS, ускорение, обороты,
ДУТ LLS1…6, уровень топлива бак1/2, глушение GSM/GPS, CAN/OBDII (педали, нагрузка двигателя, давления,
температуры, расход/пробег суточный/общий, одометр, моточасы, двери/ремни, ТО, VIN, Check Engine). Диагностика.

**5. Заправки и сливы ⭐** (мульти-ТС) — №, ТС, время начала/окончания, событие (заправка/слив), объём л.

**6. Мультимедиа** (мульти-ТС) — фото-события: Фото, Удалить, ТС, дата/время, действие (триггер снимка).

**7. Нарушения** (мульти-ТС) — №, ТС, дата/время, нарушение, параметры. Типы: превышение порога скорости
в геозоне, остановка >60с, стоянка >300с, превышение оборотов, сбой/окончание сбоя ДУТ.

**8. Посещение геозон** (мульти-ТС, нужны геозоны) — ТС, геозона, вход, выход, время в геозоне, пробег в
геозоне. Тяжёлое построение на большой выборке.

**9. Посменный отчет** (мульти-ТС, нужны смены) — по сменам: ТС, границы смены, пробег, моточасы,
расход/заправки/сливы. Тяжёлое построение.

**10. Работа группы** — детальный по сменам/суткам, группы колонок: Общие параметры/настройки, Движение и
работа, Топливо (осн.ёмкость), Топливо (доп.ёмкость). Кол.: группа ТС, дата, № смены, ТС, границы смены,
гар.номер, пробег, время работы двигателя (в т.ч. под нагрузкой), нач/кон объём, факт.расход, расчётный по
норме на 100 км, перерасход от нормы, заправки/сливы, факт.расход на 100км/моточас/час, превышение нормы на
холостом ходу; то же по доп.ёмкости.

**11. Сводный отчёт ⭐** — посуточно по ТС + «Итого», группы колонок: Общие, Движение и работа, Топливо
(осн.), Универсальный вход 1…4. Движение: пробег, общий пробег нач/кон, пробег с превышением, ср/макс
скорость, время движения, время работы двигателя (в движении/без/холостой/нормальные/предельные обороты/под
нагрузкой с %), время с выкл.двигателем. Топливо: нач/кон объём, факт.расход (всего/в движении/без/на 100км/за
моточас), заправки/сливы, превышение нормы на х.х. Универсальный вход 1…4 (доп.оборудование/ШДУТ): тип, время
работы/простоя, макс/мин/сумма, время выше/ниже допустимого, пробег с работающим доп.оборудованием.

**12. События ⭐(надмножество Нарушений)** (мульти-ТС) — №, ТС, дата/время, событие, параметры. Типы:
вход/выход геозоны, зажигание вкл/выкл, питание вкл, остановка >60с, стоянка >300с, ШДУТ вкл/напряжение,
сбой/окончание сбоя ДУТ.

**13. Статистика ⭐** — агрегаты по всей выборке (итоги/средние, без построчного перечня). Движение и работа:
сумм/ср пробег, сумм пробег с превышением, сумм/ср время движения, сумм моточасы и разбивка по режимам, время
с выкл.двигателем. Топливо: сумм/ср факт.расход (на 100км/в движении/за моточас), объём заправок, объём сливов.

**14. Стоянки** (по 1 ТС) — группы: Параметры стоянки/движения/топлива. Кол.: №, начало/окончание стоянки,
длительность, общий пробег, общий расход, объём заправок, объём сливов + итоги.

**15. Местоположение ⭐** — карта: текущие/последние позиции выбранных ТС с подписями. Без таблицы.

**16. Трек ⭐ (основной)** — карта: маршрут выбранных ТС за период с подписями. Основа всех рабочих столов.
